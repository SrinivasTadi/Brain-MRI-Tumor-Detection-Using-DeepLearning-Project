# 🧠 Brain MRI Tumor Detection

Brain MRI Tumor Detection is a deep learning-based web application that automatically analyzes brain MRI scan images to determine whether a tumor is present. The project utilizes a Convolutional Neural Network (CNN) developed with TensorFlow and Keras to learn patterns from MRI images and perform binary classification with high efficiency.

The application provides an intuitive web interface built with Streamlit, allowing users to upload MRI scan images and receive instant predictions along with a confidence score. It demonstrates the complete machine learning pipeline, including data preprocessing, model training, model evaluation, and deployment in a user-friendly interface.

This project showcases the practical application of deep learning in medical image analysis and serves as an educational implementation of computer vision techniques for healthcare.


## ✨ Features

- Upload Brain MRI scan images
- Detect the presence of a brain tumor
- Display prediction with confidence score
- Simple and interactive Streamlit web interface
- Fast image preprocessing and classification
